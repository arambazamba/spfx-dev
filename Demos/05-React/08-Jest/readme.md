# React Jest Testing

Uses [jest-preset-spfx-react16](https://www.npmjs.com/package/@voitanos/jest-preset-spfx-react16)

Jest Installation:

```
npm install jest @voitanos/jest-preset-spfx-react16 --save-dev
npm install jest@25.5.4 --save-dev --save-exact
```

Test execution:

```
npm run test
```
